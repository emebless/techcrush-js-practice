JavaScript Fundamentals Mini Projects

Overview
This project is a collection of beginner-friendly JavaScript exercises designed to help you understand core programming concepts such as variables, functions, arrays, objects, loops, and basic problem-solving.

What You Will Learn
- Variables (let, const)
- Functions (regular and arrow functions)
- Arrays and methods (push, shift, map)
- Objects and properties
- Conditional statements (if / else if / else)
- Loops (for loop)
- String methods (trim, toUpperCase, slice)
- Template literals
- Basic problem solving

Project List
1. Personal Details Logger
- Declare variables for name, age, and student status
- Log values to console

2. Arrow Function – Multiply
- Function multiply(a, b)
- Returns product of two numbers

3. Greeting Function
- greetUser(name)
- Returns: Welcome to TechCrush, [name]!

4. Array Manipulation (Fruits)
- push() adds item
- shift() removes first item
- Display updated array and length

5. Double Numbers (map)
- Use map() to double values
- Original array remains unchanged

6. Student Profile Object
- Create object with name, age, country, hobby
- Function to display profile

7. Format Name Function
- Use trim(), toUpperCase(), slice()
- Format: LASTNAME, Firstname

8. Grade Calculator
- gradeScore(score)
- Returns grade A–F

9. Multiplication Table
- for loop from 1 to 10 for number 5

10. Student Summary Project
- Array of student objects
- Use map() and gradeScore() to summarize

How to Run
Option 1: Browser Console
- Open browser → Inspect → Console
- Paste and run code

Option 2: Node.js
- Install Node.js
- Run: node filename.js

Goal
To build strong JavaScript fundamentals through simple and practical exercises.
